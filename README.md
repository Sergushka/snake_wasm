## Requirements
- [`rust-toolchain`](https://www.rust-lang.org/tools/install)
- [`trunk`](https://trunkrs.dev/)


## Steps : 
### For building :
```console 
cargo install trunk wasm-bindgen-cli
```
```console 
rustup target add wasm32-unknown-unknown
```
```console 
trunk serve
```

### For deployment :
Enable Github Pages
